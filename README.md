## 🧭 System Design Roadmap

A practical roadmap to go from **beginner → advanced system designer**:

---

### 🟢 Phase 1: Foundations
- Big-O Complexity
- Networking Basics (HTTP, TCP/IP)
- OS Basics (Threads, Memory)
- Databases (SQL vs NoSQL)

📘 Recommended:
- Algorithms Notes  
- Database Internals  

---

### 🔵 Phase 2: Core System Design Concepts
- Scalability (Vertical vs Horizontal)
- Load Balancing
- Caching (Redis, CDN)
- CAP Theorem
- Consistency Models

📘 Recommended:
- Designing Data-Intensive Applications  
- Grokking the System Design Interview  

---

### 🟡 Phase 3: Architecture Patterns
- Monolith vs Microservices
- Event-Driven Architecture
- CQRS & Event Sourcing
- API Gateway Pattern

📘 Recommended:
- Clean Architecture  
- Software Architecture Patterns  

---

### 🔴 Phase 4: Distributed Systems Deep Dive
- Message Queues (Kafka, RabbitMQ)
- Stream Processing
- Distributed Transactions
- Observability (Logs, Metrics, Tracing)

📘 Recommended:
- Kafka Streams in Action  
- Enterprise Integration Patterns  
- Production-Ready Microservices  

---

### ⚫ Phase 5: Real-World System Design
- Design:
  - URL Shortener
  - Uber / Ride Sharing
  - Netflix / YouTube
  - Chat System (WhatsApp)

📘 Recommended:
- Advanced Microservices  
- Microservices in Action
